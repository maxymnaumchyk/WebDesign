Криниця

Крини́ця або коло́дязь (так можуть називати глибоку криницю) — обладнане водне (або інше, наприклад, мінералізованої води, розсолу, нафти тощо) джерело, глибоко викопана й захищена від обвалів яма для добування води тощо з водоносних шарів землі.

У більшості сіл та містечок без водопроводу криниці є основним джерелом питної води.
